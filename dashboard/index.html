Telemetry Memory Histograms

The aggregator now records per-component memory histograms for mempool, storage, and compute subsystems. Use /telemetry on the metrics-aggregator service or the aggregator telemetry CLI command to inspect the latest percentile snapshots.

  • storage_put_object_seconds – end-to-end latency for storage uploads.
  • parallel_execute_seconds – batch execution time for the parallel executor.
  • telemetry sampling rate – adaptive sampling auto-tunes based on exporter lag.
