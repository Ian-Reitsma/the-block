Telemetry Memory Histograms

The aggregator now records per-component memory histograms for mempool, storage, and compute subsystems. Use /telemetry on the metrics-aggregator service or the aggregator telemetry CLI command to inspect the latest percentile snapshots.

  • storage_put_object_seconds – end-to-end latency for storage uploads.
  • parallel_execute_seconds – batch execution time for the parallel executor.
  • telemetry sampling rate – adaptive sampling auto-tunes based on exporter lag.

Advertising Settlement Snapshot

Explorer ingests now stream the USD totals and oracle prices that back each advertising block. Scraped Prometheus metrics expose the latest conversion inputs so dashboards and CI artefacts can graph the dual-token flows without parsing JSON payloads.

  • explorer_block_payout_ad_usd_total{peer="<explorer>"} – most recent advertising spend in micro USD reported by that explorer instance.
  • explorer_block_payout_ad_settlement_count{peer="<explorer>"} – number of settlements folded into the USD tally, enabling rate panels in CI.
  • explorer_block_payout_ad_ct_price_usd_micros / ..._ad_it_price_usd_micros – the CT and IT oracle snapshots that priced the block.

Pair these metrics with ad_readiness_total_usd_micros, ad_readiness_settlement_count, and the readiness utilization payload from ad_market.readiness to audit demand, supply, and pricing inputs end to end.
